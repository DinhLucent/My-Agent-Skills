# 🚀 Agent Onboarding — Checklist

> Complete all items before starting your first task.

---

## 1. Identity
- [ ] Read `manifest.yaml` → confirm your Agent ID
- [ ] Read your Persona file → understand responsibilities + boundaries

## 2. Skills (in order)
- [ ] **`Skills/Global/task-hub/SKILL.md`** — Hub Protocol (READ FIRST)
- [ ] `Skills/Global/.agents/rules/security.md` — Security rules
- [ ] All skills listed under your `skills:` in manifest
- [ ] `Skills/Global/Evolution/SKILL.md` — Self-learning

## 3. Context
- [ ] Read `DASHBOARD.md` → Quick Context + current progress
- [ ] Read `.hub/backlog.yaml` → find tasks matching your role
- [ ] Read `OPERATING_RULES.md` → communication + escalation rules

## 4. Ready
- [ ] Found matching task → Claim per Hub protocol
- [ ] No matching task → Tell User: "No tasks for my role"
- [ ] Wrong role task → **Refuse**

---

**Remember:** Diamond Standard output. Update Dashboard after every task. Respect other Agents' domains.
